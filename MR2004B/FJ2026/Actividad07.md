# Actividad 7: Teclado Matricial y Display 7 Segmentos ⌨️🔢

## 🎯 Objetivo

Programar el microcontrolador de 4 bits para leer un **teclado matricial** y mostrar en un **display de 7 segmentos** el dígito correspondiente a la tecla presionada.

## 📝 Instrucciones para los estudiantes

Utiliza el simulador del microcontrolador de 4 bits para desarrollar un programa que permita:

1. Leer la tecla presionada en un **teclado matricial**.
2. Identificar correctamente el dígito seleccionado.
3. Mostrar el número correspondiente en un **display de 7 segmentos** con codificador.
4. Verificar que cada tecla numérica funcione correctamente.

## 💡 Consideraciones

- El programa debe detectar correctamente la tecla presionada.
- El display debe mostrar únicamente el dígito correspondiente.
- Se debe evitar que el display muestre valores incorrectos.
- En caso de errores, analiza línea por línea el comportamiento del microcontrolador.

## 📚 Entregables

- Código fuente en archivo Excel.
- Archivo binario generado.
- Simulación funcional en Proteus.
- Video mostrando el sistema funcionando.

## ✅ Criterios de evaluación

| Criterio | Descripción |
|---|---|
| Lectura del teclado | El programa identifica correctamente la tecla presionada. |
| Visualización en 7SEG | El display muestra el dígito correcto. |
| Funcionamiento en Proteus | La simulación opera sin errores. |
| Código fuente | El código está ordenado y bien estructurado. |
