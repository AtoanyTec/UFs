# Propuesta de Proyecto COIL: Estación Biomédica Portátil para la Evaluación del Riesgo Cardiovascular

## Descripción general

Las enfermedades cardiovasculares representan una de las principales causas de mortalidad a nivel mundial. La identificación temprana de factores de riesgo permite implementar estrategias preventivas que contribuyen a reducir la incidencia de estas enfermedades y mejorar la calidad de vida de la población. En este contexto, se propone el desarrollo de un proyecto COIL (Collaborative Online International Learning) entre estudiantes de Medicina e Ingeniería Mecatrónica para diseñar e implementar una **Estación Biomédica Portátil** orientada a la evaluación del riesgo cardiovascular.

El proyecto consiste en el desarrollo de una plataforma capaz de recopilar, integrar y analizar variables fisiológicas, antropométricas, bioquímicas y de estilo de vida asociadas al riesgo cardiovascular. La estación permitirá registrar información como tensión arterial, niveles de colesterol total, LDL y HDL, perímetro abdominal, peso, índice de masa corporal (IMC), sexo, actividad física, tabaquismo, antecedentes familiares, consumo de alcohol y hábitos de alimentación.

## Variables consideradas

| Variable | Método de captura |
|-----------|-----------|
| Tensión arterial | Captura manual mediante tensiómetro comercial |
| Colesterol total | Captura manual en software |
| LDL | Captura manual en software |
| HDL | Captura manual en software |
| Perímetro abdominal | Captura manual en software |
| Peso | Báscula electrónica desarrollada con Arduino |
| IMC | Calculado automáticamente por software |
| Sexo | Captura manual en software |
| Actividad física | Captura manual mediante cuestionario |
| Tabaquismo | Captura manual mediante cuestionario |
| Herencia (antecedentes familiares) | Captura manual mediante cuestionario |
| Consumo de alcohol | Captura manual mediante cuestionario |
| Tipo de alimentación | Captura manual mediante cuestionario |

## Participación de los estudiantes de Medicina

Los estudiantes de Medicina aportarán los conocimientos clínicos y bioquímicos necesarios para comprender la relevancia de cada variable asociada al riesgo cardiovascular. Sus responsabilidades incluirán:

- Diseñar los instrumentos de recolección de datos clínicos.
- Justificar la importancia de cada variable desde una perspectiva médica y bioquímica.
- Interpretar los resultados obtenidos.
- Definir los criterios clínicos para la clasificación del riesgo cardiovascular.
- Diseñar y validar el algoritmo de evaluación del riesgo cardiovascular.
- Establecer los rangos que permitan clasificar a los individuos en categorías de riesgo bajo, moderado o alto.

## Participación de los estudiantes de Ingeniería Mecatrónica

Los estudiantes de Ingeniería Mecatrónica serán responsables del diseño e implementación de la solución tecnológica. Sus responsabilidades incluirán:

- Diseñar y construir una báscula electrónica basada en Arduino para la medición automática del peso corporal.
- Desarrollar la interfaz de captura de datos clínicos y biométricos.
- Diseñar la arquitectura de software para el almacenamiento y procesamiento de la información.
- Implementar computacionalmente el algoritmo de evaluación definido por los estudiantes de Medicina.
- Diseñar dashboards y reportes para la visualización de resultados.
- Integrar el hardware y software en una estación biomédica funcional.

## Producto final

Como resultado del proyecto, los equipos interdisciplinarios desarrollarán una estación biomédica portátil capaz de:

1. Capturar variables relacionadas con el riesgo cardiovascular.
2. Calcular automáticamente indicadores derivados, como el IMC.
3. Integrar datos clínicos, biométricos y de estilo de vida.
4. Aplicar un algoritmo de evaluación del riesgo cardiovascular definido por los estudiantes de Medicina.
5. Clasificar a cada individuo en categorías de riesgo bajo, moderado o alto.
6. Generar un reporte o dashboard con los resultados obtenidos.

## Impacto esperado

Este proyecto permitirá que los estudiantes trabajen en equipos interdisciplinarios para resolver un problema real de salud mediante el desarrollo de una solución tecnológica basada en evidencia científica. La colaboración entre Medicina e Ingeniería Mecatrónica favorecerá el intercambio de conocimientos, el desarrollo de competencias profesionales y la comprensión de cómo la tecnología puede apoyar la prevención y el monitoreo de enfermedades cardiovasculares.
